Shodan.UI

What's this?

Shodan UI is a small library of common User Interface components powered by the PrototypeJS and Scripty2 frameworks.

Current components

Shodan UI currently comprises the following components:

  • Slidey panel
  • Accordion
  • Slideshow

More coming soon

  • Scroller Slideshow
  • Lightbox
  • Carousel
  • Tabs
  • Auto complete
